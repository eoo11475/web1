td= table data 의 약자 즉 표 각각의 실제 데이터 tr태그는 그사이에 있는 td행을 같은 행으로 표시 td와 tr 태그를 table태그로 전체 감싸줌. 태그 table border= "1" 은 테두리를침 thead는 테이블의 머리 tbody는 본문 tfoot는 테이블의 마지막 th는 각 셀의 글자를 굵게 표시 rowspan= "2"이 태그가 포함된 세로가 병합됨. 그 아래 행 대전 태그 는 지움. colspan ="3"는 3개의 가로축 셀이 합병됨.
